# Worktrees

Commit+ lets you create, switch, and remove **git worktrees** from the sidebar — perfect for isolated parallel feature work without leaving your main checkout behind.

## Create a worktree

1. Right-click a branch (or use the worktree section) and choose **New Worktree**.
2. Pick a branch and a path.
3. Commit+ creates the worktree and opens it in a new window.

## Remove a worktree

Select a worktree in the sidebar and choose **Remove**. Commit+ cleans up the working tree and its metadata.
