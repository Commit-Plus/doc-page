# Cherry-Pick & Revert

## Cherry-pick

Drag a commit from one branch onto another to **cherry-pick** it. Commit+ applies the commit's changes onto the target branch.

## Revert

Hold `⌥` while dragging a commit to **revert** it — creating a new commit that undoes the original change.

Both actions register undo entries, so you can roll them back with `Cmd+Z`.
