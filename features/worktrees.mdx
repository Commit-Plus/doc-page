# Smarter Worktree Management

Manage **git worktrees** visually from the sidebar for isolated parallel feature work — keep your main checkout untouched while you work on something else.

- **Create** a worktree from any branch, opening it in a new window.
- **Switch** between worktrees from the sidebar.
- **Remove** a worktree, cleaning up the working tree and its metadata.
