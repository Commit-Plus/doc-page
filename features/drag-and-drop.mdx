# Drag & Drop

Complex Git actions become effortless with drag and drop in Commit+.

## Commits

- **Reorder / Squash / Fixup** — drag commits in the history view to launch an interactive rebase.
- **Cherry-Pick** — drag a commit from one branch to another.
- **Revert** — hold `⌥` while dragging a commit to revert it.

## Branches

- **Merge** — drag a branch onto HEAD.
- **Rebase** — hold `⌥` while dragging a branch onto HEAD.
- **Publish / Pull** — drag a branch to/from the remote section of the sidebar.

## Files & Stashes

- **Stage / Unstage** — drag files between working copy and staged views.
- **Stash files** — drag files to the stash view.
- **Apply** — drag a stash or individual file back to the working copy.
