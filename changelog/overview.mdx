## New features

**Pull Requests view** — Browse open pull requests for a connected repository without leaving Commit+. Each entry shows the PR number and title, source and target branches, author, state (open, closed, merged, or draft), and last updated time. Use **Open in Browser** to jump to the provider's PR page. GitHub is supported at launch through a connected provider account. See [Pull Requests](/features/pull-requests).
