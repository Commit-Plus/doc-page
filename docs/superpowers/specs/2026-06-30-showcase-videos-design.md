# Showcase Videos Design

## Goal

Add provided showcase videos to the relevant feature pages without inspecting video content. Use each filename as the source of meaning and description.

## Mapping

- `showcase/drag-commit-cherry-pick.mp4` goes on Drag and Drop.
- `showcase/drag-to-create-new-branch.mp4` goes on Drag and Drop.
- `showcase/undo-checkout-branch.mp4` goes on Undo Action.

## Playback

Videos must autoplay, stay muted, loop forever, play inline, and omit controls. The page should not expose video controls.

## Verification

Run Mintlify validation, broken-link checks, and git diff whitespace checks.
