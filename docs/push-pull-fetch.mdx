# Push, Pull & Fetch

| Action | Shortcut | Description |
|--------|----------|-------------|
| Pull | `Cmd+Shift+P` | Fetch and fast-forward / merge the current branch |
| Push | `Cmd+Option+P` | Push the current branch to its upstream |
| Fetch | `Cmd+Option+F` | Update remote-tracking refs without merging |

## Drag to remote

Drag a local branch into the **Remotes** section of the sidebar to **publish** it. Drag a remote branch back to publish... wait, drag a remote branch onto a local branch to **pull** it.

## Undoing remote actions

Pull and publish can be undone with `Cmd+Z`. See [Undo Any Action](/features/undo-any-action).
