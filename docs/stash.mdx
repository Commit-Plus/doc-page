# Stash

`Cmd+Shift+S` saves your working copy and staged changes into a new stash.

## Drag & drop

- Drag files **to the stash view** to stash only those files.
- Drag a stash (or an individual file from a stash) **back to the working copy** to apply it.

## Undo

Stash save, drop, apply, and pop are all undoable with `Cmd+Z`. See [Undo Any Action](/features/undo-any-action).
