# Branch, Merge & Rebase

## Branch

`Cmd+Shift+B` creates a new branch from the current HEAD. Switch branches from the sidebar or via Quick Search (`Cmd+Shift+F`).

## Merge

Drag a branch onto **HEAD** to merge it into your current branch. Alternatively use `Cmd+Shift+M`.

## Rebase

Hold `⌥` while dragging a branch onto HEAD to rebase onto it instead of merging.

## Interactive rebase

Drag commits within the history view to reorder, squash, or fixup. Commit+ launches an interactive rebase and resolves it for you automatically.
