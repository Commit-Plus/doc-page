# Conflict Resolution

When a merge, rebase, cherry-pick, or revert produces conflicts, Commit+ opens the **conflict resolution** view.

## What you get

- A visual diff viewer with inline conflict markers (`<<<<<<<`, `=======`, `>>>>>>>`).
- Stage-resolution controls to accept ours, theirs, or both per hunk.
- Navigation between unresolved conflicts.
- The ability to **abort** the merge, rebase, cherry-pick, or revert and return to your prior state.

## Undo

Supported aborts and resolved history actions can register undo entries. Use `Cmd+Z` to roll back when an undo entry is available.
