AI features are coming soon. The first focus is practical Git assistance: better commit messages and safer conflict resolution, not a general chatbot bolted onto the app.

<CardGroup cols={2}>
  <Card title="Generate commit message" icon="sparkles" badge="Coming soon">

  </Card>

  <Card title="Suggest conflict resolution" icon="git-merge" badge="Coming soon">
    Suggest merge-conflict resolutions while keeping the final choice visible and user-controlled.
  </Card>
</CardGroup>

## Scope

- Commit-message generation should work from the staged diff.
- Conflict suggestions should explain the proposed resolution before applying it.
- AI output should remain editable and reviewable.

## Privacy direction

Commit+ does not currently ship AI features. Any future AI workflow should make data flow explicit before sending repository content outside the app.
